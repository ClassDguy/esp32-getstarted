Set up computer

  • Serial driver for Adafruit Huzzah32
    • https://www.silabs.com/products/development-tools/software/usb-to-uart-bridge-vcp-drivers
    • Install the legacy version & reboot - you’ll need to open system preferences to allow install to complete
  • Tools to work with ESP32 (you may need to use sudo)
    • pip3 install esptool
    • pip3 install adafruit-ampy
  • Micropython software binary
    • https://micropython.org/download/#esp32

Set up ESP32

  • Connect device by USB
  • Check the name of serial port: ls /dev/tty*
  • Erase the device
    •  esptool.py --chip esp32 -p /dev/tty.SLAB_USBtoUART erase_flash
  • Flash the micropython software
    •  esptool.py --chip esp32 -p /dev/tty.SLAB_USBtoUART write_flash -z 0x1000 esp32-20180308-v1.9.3-407-g250b24fe.bin
  • Reboot device
  • Try out the REPL
    •  screen /dev/tty.SLAB_USBtoUART 115200
    •  help()
    •  print("hello")
  • List files
    •  import os
    •  os.listdir()
  • Ctrl-A Ctrl-\ to get out of screen mode
  • See files on the device disk
    •  ampy -p /dev/tty.SLAB_USBtoUART ls

Coding Workflow

  • Write code
    •  Use your text editor to create python code
    •  Save as .py file
    •  Sample code: blink.py
  • Put or get a file on the device
    •  Note: You must exit the "screen" mode before you can use amp to access device files
    •  cd /directory/where/your/code/is
    •  ampy -p /dev/tty.SLAB_USBtoUART put blink.py
    • ampy -p /dev/tty.SLAB_USBtoUART get blink.py
  • Start the REPL
    •  screen /dev/tty.SLAB_USBtoUART 115200
  • Run the python code on the device
    •  import blink.py
  • Exit the REPL
    • Ctrl-A Ctrl-\
  • Repeat
